# ttPin MVP架构简化说明

## 架构调整（2026-01-08）

为了快速上线第一版本，项目架构进行了重大简化：

### ❌ 移除的组件

1. **后端API服务器**（apps/backend）
   - 用户认证系统
   - JWT token管理
   - 配额跟踪
   - 翻译代理API
   - 生词本云同步

2. **管理员后台**（apps/admin）
   - 用户管理
   - VIP设置
   - 配额调整
   - 统计dashboard

3. **数据库服务**
   - PostgreSQL
   - Prisma ORM
   - 云端数据存储

4. **Azure Communication Email**
   - 邮箱验证码
   - 用户注册

### ✅ MVP保留的核心功能

1. **Tauri桌面应用**（apps/desktop）
   - ✅ 快捷键唤出/隐藏
   - ✅ 窗口管理（右下角、置顶、可拖拽）
   - ✅ 翻译UI（语言选择、输入输出）
   - ✅ 中英文i18n切换
   - ✅ 设置界面（配置Azure密钥）

2. **Azure直接调用**
   - ✅ Azure Translator LLM（用户自己的密钥）
   - ✅ Azure Text-to-Speech（用户自己的密钥）
   - 密钥存储在OS原生安全存储（Windows凭据管理器）

3. **本地数据存储**
   - ✅ SQLite本地数据库
   - ✅ 生词本（单词+翻译+例句）
   - ✅ 用户设置（Azure配置）
   - ✅ 支持导出CSV/JSON

## 新架构图

```
┌────────────────────────────────────────────┐
│     Desktop App (Tauri + React)            │
│                                            │
│  ┌──────────────────────────────────────┐  │
│  │  Frontend (React + i18n)             │  │
│  │  - 翻译UI                             │  │
│  │  - 生词本管理                         │  │
│  │  - 设置界面（Azure配置）              │  │
│  └──────────────────────────────────────┘  │
│                                            │
│  ┌──────────────────────────────────────┐  │
│  │  Tauri Rust Backend                  │  │
│  │  - 全局快捷键                         │  │
│  │  - 窗口管理                           │  │
│  │  - OS安全存储（密钥管理）             │  │
│  │  - SQLite本地数据库                   │  │
│  └──────────────────────────────────────┘  │
│                                            │
│          ↓ 直接调用                        │
│                                            │
│  ┌──────────────────────────────────────┐  │
│  │  用户的Azure资源                      │  │
│  │  - Translator API (LLM)              │  │
│  │  - Text-to-Speech API                │  │
│  └──────────────────────────────────────┘  │
└────────────────────────────────────────────┘
```

**特点**：
- ✅ 无需服务器
- ✅ 无需数据库维护
- ✅ 无需用户注册
- ✅ 完全离线工作（配置后）
- ✅ 数据隐私（本地存储）

## 技术栈简化

| 原架构 | MVP架构 | 说明 |
|--------|---------|------|
| Fastify后端 | ❌ 移除 | 无需服务器 |
| PostgreSQL | ❌ 移除 | 改用SQLite |
| JWT认证 | ❌ 移除 | 无需登录 |
| Azure Email | ❌ 移除 | 无需注册 |
| Prisma ORM | ❌ 移除 | 使用Tauri SQL插件 |
| 管理后台 | ❌ 移除 | 无用户管理 |
| Tauri桌面 | ✅ 保留 | 核心应用 |
| React + TS | ✅ 保留 | UI框架 |
| i18next | ✅ 保留 | 国际化 |
| Azure Translator | ✅ 保留 | 翻译引擎 |
| Azure TTS | ✅ 保留 | 语音合成 |

## 优势分析

### MVP简化架构的优势

1. **开发速度快**
   - 无需后端开发
   - 无需数据库部署
   - 专注核心功能

2. **部署成本低**
   - 无需服务器托管（$0/月）
   - 无需数据库维护（$0/月）
   - 只需GitHub Pages托管文档

3. **用户隐私好**
   - 数据完全本地存储
   - 无需上传个人信息
   - 用户控制自己的密钥

4. **扩展性强**
   - 未来可随时添加后端
   - 可选择性添加云同步
   - 架构升级不影响核心功能

### 潜在限制

1. **生词本无法跨设备同步**
   - 解决方案：支持导出/导入
   - 未来：可选云同步服务

2. **需要用户自备Azure账号**
   - 解决方案：提供详细配置指南
   - 未来：可提供开发者密钥选项

3. **无使用统计**
   - 影响：无法追踪用户量
   - 未来：可选匿名统计
